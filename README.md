**Dataset:** Loan Prediction Dataset (Kaggle)

**Results:** Achieved 78.86% accuracy. Credit history is the key factor in loan approval.

## Task 3: Customer Churn Prediction
**Objective:** Identify customers likely to leave the bank.
**Dataset:** Churn Modelling Dataset (Kaggle)
**Approach:** Encoded categorical features, trained Random Forest Classifier, analyzed feature importance.
**Results:** Age is the strongest predictor of churn, followed by EstimatedSalary and CreditScore.

## Task 4: Predicting Insurance Claim Amounts
**Objective:** Estimate medical insurance charges based on personal data.
**Dataset:** Medical Cost Personal Dataset (Kaggle)
**Approach:** Trained Linear Regression model, visualized impact of BMI, age, and smoking status.
**Results:** MAE: 4186.51, RMSE: 5799.59. Smoking status has the highest impact on charges.

## Task 5: Personal Loan Acceptance Prediction
**Objective:** Predict which customers are likely to accept a loan offer.
**Dataset:** Bank Marketing Dataset (Kaggle)
**Approach:** Explored features like age, job, marital status, trained Decision Tree Classifier.
**Results:** Achieved 76.3% accuracy. Duration of contact is the key factor in loan acceptance.
